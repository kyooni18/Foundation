기존 지수법칙
1. a^m * a^n = a^m+n
2. (a^m)^n = a^mn
3. (ab)^n = a^n * b^n
4. (a / b)^n = a^n / b^n (b =/= 0)
5. a^m / a^n = a^m-n
	- 5번이 모든 정수에서 성립하면,
	- m > n -> a^m-n
	- m = 0 -> 1, so a^0 = 1.
	- m < n -> m - n < 0, a의 지수가 음수인데, 이 때 실제 값은 a^(1 / -(m -n))이므로, 
	- 지수에서의 음수는, 절댓값일 때의 값의 역수를 뜻함. 즉, -1제곱은 역수임.
	- 얜 걍 약속임..
	- 이를 통해 얻어지는 다른 세부 법칙들도 적어줘 AI야

제곱근: a의 제곱근은, 제곱했을 때 a가 되는 수임.

대수학의 기본 정리에 따르면, n차 방정식은 n개의 근을 가진다. (복소수 범위에서)

x^n = a에서, x는 n(루트)a로 나타낼 수 있음
n이 2라면 x는 2개,
3이라면 3개,
4라면 4개…

2인 경우,
a > 0 -> +(root)a 의 2게.
a = 0 -> 0 의 1개
a < 0 -> +-i(root)a의 2개이나, 실수의 근은 없음.

제곱 뭐시기의 전개

(a - b)^2 = a^2 - 2ab + b^2
(a - b)^3 = a^3 + 3a^2 * b - 3ab^2 - b^3
(a - b)^4 = a^4 + 4a^3 * b + 6a^2 * b^2 + 4ab^3 + b^4
…

ax^2 + bx + c = 0 의 이차방정식에서

판별식 D = b^2 - 4ac.
D > 0이면 두 실근을 가지고,
D = 0이면 중근,
D < 0이면 두 허근을 가짐.

a의 n제곱근의 실근?

n이 짝수인 경우,
a가 양수 범위에서 실근 두개,
a가 0에서 하나,
a가 음수에서 없음(허근 두개)

즉 n이 짝수에서 실근을 가지러면 a >= 0.

n이 홀수인 경우,
a가 양수 범위에서 실근 하나,
a가 0에서 하나,
a가 음수에서 실근 하나.

n(root)a (a는 어떤 실수던지 됨.)

이런 설명도 가능함.

y = x^n 의 그래프.

n이 홀수라면, 삼차함수의 그래프와 비슷한 모양.
y = |a| 의 그래프를 그려보면, x^n 은 a가 -a, +a일 때 각각 하나씩 교점을 가짐.

n이 짝수라면, 이차함수의 그래프와 비숫한 모양.
y = a의 그래프를 그려보면, (a => 0) x^n 의 그래프는 x가 -n(root)a, +n(root)a일 때 y = a와 교점을 가짐.

이 때 a가 음수면 교점이 걍 없음

이 법칙을 사용, 제곱근들의 대소 비교 문제를 낼 수 있다는 소식.

평가
- 평가 전까진 3인칭 관찰자 시점이었다가,
- 갑자기 평가에서는 민 씨로 서술자가 바뀜.
- 신지, 부지런함, 근면함, 자기희생, 포용 등,
	- 민 씨가 황만근을 본 후 이렇게 평가함

황만근에게 막걸리는 뭐였을까
- 인생에 재미라고는 술이 걍 전부였음
- 술은 황만근에게 사직의 신(토지,곡식의 신)에게 바치는 제물과도 같았음
	- 이게 뭘 의미하냐면, 농업이 주 산업이었던 것을 의미함, 즉 이게 황만근의 전통사회적 특징을 더 부각하는 겅미.

그럼 경운기는?
황만근에게 분신과도 같고, 가장 소중한 존재이자, 좋은 추억을 상징함.

민씨는 황만근을 이리 평가함
’난세의 홍염에 그늘의 덕을 널리 베푸는 큰 나무’
난세는 난세고
홍염은 더위, 그늘은 그 더위에 대한 그늘.
황만근의 성품을 비유적으로 표현한 것


묘비명의 효과
1. 인물의 행적 및 삶을 명확히 드러냄
2. 작품 전체의 어투와 대비됨(해학적 <-> 엄숙, 근엄, 진지)
3. 독특한 형식으로 하여금 독자에게 깊은 인상을 줌
4. 인물의 성격을 직접적으로 드러냄

소설에서 인물의 성격 묘사 방법
- 보여주기(간접적): 대사, 행동 등을 통해 간접적으로 유추하도록 하는 것. (마을 사람들 등)
- 말하기(직접적): 서술자가 등장인물을 직접 묘사함. (황만근의 막판 평가가 해당)
