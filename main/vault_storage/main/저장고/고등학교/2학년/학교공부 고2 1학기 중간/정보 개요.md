# **출력 관련 함수**

print()

괄호 안의 것을 출력

문자열이 아니어도 됨

예시

- print("asdasd: %.2f" %변수")라면 변수를 소수 둘째자리까지 반올림해서 문자열 안에 포함
- print("{} {} {}".format(변수1, 변수2, 변수3)) 변수1 2 3을 각각 순서대로 문자열에 넣음
- print("{2}, {1}, {0}".format(변수1, 2, 3)) 변수 1 2 3을 {} 안에 지정한 인덱스로 집어넣음
- print("asdasd", end = "as") end의 기본값은 줄바꿈 출력 결과는 “asdasd” + “as”가 됨

1. 유니코드면 가능
2. 변수명 맨 앞에 숫자는 안됨
3. _ 를 제외한 특수문자는 안됨(공백 포함) 3-1. _는 맨 앞에 나와도 ㄱㅊ

# **turtle 라이브러리**

turtle

그리는 라이ㅡㅂ러리

다음과 같이 사용 가능:

- t = turtle.Turtle(),
- t = turtle.Pen()
- import turtle as t 모두 같은 표현

초기 방향은 우측, 좌표는 (0, 0)

색상 형식

기본 저장된 색의 이름, Hex 값, (r, g, b) 중 하나

함수들

- forward() : 앞으로
- goto(x, y) : 해당 좌표로 직선으로 가면서 선을 그림
- right() / left() : 해당하는 방향으로 입력된 각도만큼 돎 초기 상태에서 right(120)은 회전이 진행되는 구간에서 원 각도로 연장선을 그은 뒤, 우측으로 120도만큼 돈 값임.
- pensize() : 선 굵기 결정
- hideturtle() : 그리는 놈 숨김
- penup() / pendown() : 그릴지 말지를 결정
- circle(반지름) : 반시계 방향으로 원을 그림 반지름이 음수라면 시계 방향으로 그림
- color() : 색상 결정
- begin_fill() / end_fill() : 그려지는 도형을 채우거나 안 채움
- shape() : 그리는 놈 모양

# 배열

[]

어떤 변수형이던 됨

안에 배열을 넣을 수도

인덱싱

- [숫자] 숫자 번째의 항목을 가져옴. 단, 1부터가 아닌 0부터임
- [숫자:숫자2] 숫자 번째부터 숫자2 보다 작은 번째까지를 가져옴

range()

range(시작, 끝, 간격)

시작부터 끝보다 작은 숫자들을 가져옴

간격은 기본 1, 숫자들 사이의 간격(이전 숫자에 더해서 다음 숫자)

시작도 생략 가능, 0부터 끝보다 작은 것까지

# **문자열 관련**

문자열 인덱싱 및 슬라이싱

- (문자열)[숫자] 라면 지정한 번째(배열/range랑 똑같음) 가져옴
- [?:?] 라면 ?번쨰부터 ?-1번쨰까지를 합친 걸 가져옴

특수 문자

- \t : 공백 10칸
- \n : 줄바꿈
- , : 한칸 띄고 합침
- + : 합침
- : 반복

# **리스트(배열) 조작**

예시

```python
ar = [1, "d", True, [0, 1]]
//ar[3][0]이라면
//ar의 3번 항목인 [0, 1]을 가져와서
//그 배열에서 0번 항목을 가져옴
```

항목 삭제 및 추가

- del (배열)[] 지정한 범위의 항목 삭제
- len(배열) 항목 수(정수) 출력.
- append(추가할거, 인덱스) 추가할거를 지정한 위치에 추가. 인덱스는 생략 시 맨 끝에 추가 3이라면, 3번 항목이 4번 항목으로 밀리고 추가할게 추가됨
- remove() 해당하는 값을 가진 항목을 삭제. 단, 해당하는 항목이 여러 개라면 앞의 하나만 삭제

# **조건 및 비교 연산자**

조건

- == : 같다면
- != : 같지 않다면
- > : 보다 크다면
- < : 보다 작다면 *문자열의 경우 < > 는 문자의 가나다(abc) 순서, 작은 경우가 사전상에서 앞
- (조건) and (조건2) 조건, 조건2를 모두 만족하는 경우에 True
- (조건) or (조건2) 둘 중 하나를 만족하는 경우 True
- not (조건) 조건이 거짓일 경우 True

# **조건문**

if 구문

```python
if (참/거짓):
	print("if")
else if:
	print("else if")
else:
	print("else")
```

조건이 참이라면 if 안 것을 실행하고,

if 조건에 해당하지 않으나 else if 조건에 해당하면 else if를 실행,

그 외 모든 경우에 대해 else 안의 것을 실행
